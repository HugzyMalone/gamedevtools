# gamedevtools.dev A collection of free browser-based tools for indie
game developers. Monetised via display ads. Built with React + Vite +
Tailwind CSS. ## Stack - React 18 + Vite - Tailwind CSS v3 - React Router
v6 (client-side routing) - Recharts (for tool visualisations) - Deployed
on Vercel (static) ## Structure - / = Homepage with tool card grid 
/tools/loot-table-simulator - /tools/damage-formula-sandbox 
/tools/xp-curve-designer - /tools/palette-generator 
/tools/game-jam-ideas - /tools/sprite-sheet-slicer ## Design - Dark theme
with game-dev aesthetic - Responsive (mobile-first) - Each tool page: H1
title, intro paragraph, interactive tool, related tools links - Homepage:
grid of tool cards with icon, title, description, link
